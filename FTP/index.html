DATA_GLOSSARY

RETR The FTP command used to retrieve a copy of a file from the server to the client.
PASV Mode Passive mode where the server "listens" for a data connection from the client, often used to bypass firewalls.
Base64 A binary-to-text encoding scheme used to represent data in an ASCII string format.
Telemetry The collection of data from remote points for monitoring and analysis.
